# CJInstant
## 计时类应用
#### 类似备忘录，还可以显示到下次节日还剩多少时间，
#### 未发生的事情是显示还剩xx天，一般是提醒自己要去做
#### 已发生的事情是显示已过xx天，一般是提醒自己什么纪念日

##### "全": 全部记录
##### "周": 以今天为原点，到7天后
##### "月": 以今天为原点，到30天后
##### "昔": 已经过去的
